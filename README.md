# PassStrength

A cross-platform CLI cybersecurity tool built with Python to analyze password strength and generate secure passwords.

## Features

- **Password Strength Checker**: Analyzes length, character variety, and common password lists.
- **Weak Password Examples**: Educational section on why certain passwords are insecure.
- **Strong Password Generator**: Uses Python's `secrets` module for cryptographically secure generation.
- **Stylish CLI**: Colorful and user-friendly terminal interface.
- **Cross-Platform**: Works on Windows, macOS, and Linux.

## Installation

1. Clone the repository or download `pass_strength.py`.
2. Install the required dependency:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the tool using Python:
```bash
python pass_strength.py
```
